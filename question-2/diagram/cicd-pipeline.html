1

🔐 Secret Scanning

  • Gitleaks — scan git history
  • TruffleHog — high entropy strings
  • Pre-commit hook (local)
  • Blocks merge on any finding
2

🔒 SAST

  • Semgrep (p/security-audit)
  • CodeQL (GitHub Advanced Security)
  • Scan only changed files (incremental)
  • Upload SARIF to GitHub Security tab
3

📦 SCA

  • Trivy filesystem scan
  • Snyk for dependency audit
  • License compliance check
  • CRITICAL CVE → block merge
4

🐳 Build & Scan

  • Docker multi-stage build
  • Distroless base image
  • Trivy image scan
  • Layer cache via GHA cache
5

📋 SBOM + Sign

  • Syft generates SPDX-JSON SBOM
  • Cosign signs image (keyless)
  • Provenance attestation (SLSA L2)
  • Stored in Rekor transparency log
6

🟢 Deploy Dev

  • ArgoCD GitOps sync
  • Helm chart + values.dev.yaml
  • Kyverno verifies image signature
  • Smoke test via GitHub Actions
7

🟡 Staging Tests

  • Full integration test suite
  • E2E tests (Playwright)
  • k6 load test (SLO verification)
  • Auto-promoted from dev
8

🔴 Production Deploy

  • Manual approval required
  • Blue/Green or Canary strategy
  • Auto-rollback on health failure
  • PagerDuty alert on degradation
