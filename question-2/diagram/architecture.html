⚙️ Services Overview

  • Frontend: React app served via Nginx, behind CloudFront CDN (prod)
  • API Service: FastAPI / Node.js — REST endpoints, connects to DB + external APIs
  • Apache Airflow: CeleryExecutor (prod), LocalExecutor (dev/staging) — workflow orchestration, DAG scheduling, data pipeline management
  • PostgreSQL: Application database + Airflow metadata DB
  • Redis: Celery broker for Airflow + API caching layer

🔒 Security Architecture

  • Kubernetes Network Policies — zero-trust, deny-by-default
  • Pod Security Standards — restricted profile (no root, read-only FS)
  • RBAC — least-privilege service accounts per service
  • mTLS for service-to-service communication (Istio/Linkerd optional)
  • External Secrets Operator — secrets synced from AWS Secrets Manager
  • Image signing (Cosign) + Kyverno admission controller for verification
  • VPN / PrivateLink for all external connections — no public DB access

📊 Observability Strategy

  • Metrics: Prometheus + Grafana — service SLOs, latency, error rate
  • Logs: Fluent Bit → Elasticsearch → Kibana — structured JSON logs
  • Tracing: OpenTelemetry → Jaeger — distributed request tracing
  • Alerting: AlertManager → PagerDuty (critical) + Slack (warning)
  • Health Checks: Liveness + Readiness + Startup probes per service
  • Dashboards: Pre-built Grafana boards — per service + infra overview

🔗 External Connections

  • VPN / AWS Site-to-Site: For on-premise database connections
  • AWS PrivateLink: For cloud-hosted external services
  • Secrets Manager: External API keys, DB credentials stored centrally
  • ConfigMap per env: Base URLs, feature flags, timeout configs
  • Connection pooling: PgBouncer for PostgreSQL (Airflow + API share pool)
  • Circuit breaker: Prevent cascade failures from external system outages
