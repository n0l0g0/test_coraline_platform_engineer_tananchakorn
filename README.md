## Quick Summary

### Question 1 — Metabase + PostgreSQL on AWS

**Tech Stack**: Terraform + AWS ECS Fargate + RDS PostgreSQL + ALB + Secrets Manager

Key highlights:
- PostgreSQL is in **private subnets with no internet route** — unreachable from public
- DB credentials generated by Terraform and stored in **AWS Secrets Manager** (never in code)
- ECS tasks fetch secrets at startup via `secrets:` field (not env vars)
- IAM roles follow **least privilege** — scoped to exact secret ARN
- **Multi-AZ RDS** with encryption at rest + Performance Insights
- **VPC endpoints** for Secrets Manager, ECR, CloudWatch — traffic never traverses internet
- ALB with HTTPS, TLS 1.3 policy, HTTP→301 redirect, access logs to S3
- ECS auto-scaling (CPU/Memory based), deployment circuit breaker with auto-rollback
- CloudWatch alarms for: CPU, Memory, 5xx errors, healthy hosts, RDS storage

**To deploy**:
```bash
cd question-1/terraform
terraform init
terraform plan -out=tfplan
terraform apply tfplan
terraform output metabase_url
```

### Question 2 — Architecture & CI/CD Pipeline

**Tech Stack**: EKS + GitHub Actions + ArgoCD + Argo Rollouts + Prometheus/Grafana + ELK

Key highlights:
- **3-environment model**: dev (auto) → staging (auto + tests) → prod (manual approval)
- **8-stage CI pipeline**: secret scan → lint → unit tests → SAST → SCA → build → image scan → SBOM + sign
- **Canary deployment**: 4 phases (header-only → 10% → 25% → 50% → 100%) with automated Prometheus AnalysisRun at every step — auto-abort if any metric breaches SLO
- **Blue/Green** for Airflow Scheduler (stateful — must have exactly 1 active scheduler)
- **GitOps**: ArgoCD syncs separate `gitops-helm-values` repo — declarative, auditable
- **Secrets**: External Secrets Operator syncs from AWS Secrets Manager to K8s Secrets
- **External connections**: On-premise via VPN, cloud services via PrivateLink
- **Observability**: Prometheus + Grafana (metrics) + ELK (logs) + Jaeger (tracing) + PagerDuty (alerts)
- **Auto-rollback**: Production health check failure triggers automatic rollback in < 5 minutes
